Run Scanner & Sync Locally

This web application is currently running in Static Mode on GitHub Pages. To scan your inbox, add subscriptions, or edit your blocklist, you need to run the application locally on your computer.

1 Start the Backend Server

Open your terminal in the project directory and start the Node.js local server:

npm start

2 Open Local Dashboard

Navigate to the local dashboard in your browser. All interactive features (Scanning, Adding, Deleting, Blocklisting) will be fully enabled:

http://localhost:3000

3 Run Scanner Manually (Alternative)

You can also execute the Gmail inbox scanner directly from your terminal:

python scan_inbox.py

To force-rescan your inbox since January 2026: python scan_inbox.py --reset

4 Publish Updates to GitHub

After scanning new items locally, commit and push changes to update your static GitHub Pages site:

git add data/subs.sqlite webapp/subscriptions.json; git commit -m "Sync subscriptions"; git push
